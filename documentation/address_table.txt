Peripheral Address Table
Derived from: BCM2835-ARM-Peripherals

* Physical peripheral addresses start at 0x20000000
* Virtual peripheral addresses start at 0x7E000000 and end at 0x7E2000B0
* Physical address 0x20000000 and virtual address 0x7E000000 are equivalent
* BCM2835-Arm-Peripherals.pdf documents register addresses starting on page 89
